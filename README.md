NearServe – Local Service Discovery and Marketplace Platform
Overview

NearServe is a full-stack web application that helps users discover nearby service providers such as plumbers, electricians, tutors, and AC repair technicians.

Features :
JWT Authentication
Role-Based Access Control
Location-Based Search
Haversine Distance Calculation
Provider Verification
Reviews and Ratings
Customer-Added Service Providers
Admin Dashboard

Tech Stack :
Frontend :
React.js
React Router
JavaScript
HTML/CSS

Backend :
Spring Boot
Spring Security
Spring Data JPA
JWT Authentication

Database :
MySQL

Architecture :
React Frontend
       ↓
Spring Boot REST APIs
       ↓
Spring Data JPA
       ↓
MySQL Database
